## Inventory System

A grid based Inventory System written in C++ in Unreal Engine 5

![](https://i.imgur.com/ZCVaVCp.gif)

### Features:

* Pick Up Items
* Drag and Drop in Inventory
* Rotate Items in Inventory
* Drop Items
